# Task 7: Fetch and Display Data from a Public API

### **Objective**
Use JavaScript Fetch API to retrieve user data from a public API and display it dynamically on a webpage.

---

##  **Features Implemented**
- Fetch API to request data from public endpoint
- JSON parsing
- Dynamic DOM rendering using JavaScript
- Professional card UI with hover effects
- Loader while fetching
- Error message handling and display
- Working Reload button
- Responsive design using CSS Grid

##  **How to Run This Project**
1. Download or copy the code into a file named `index.html`
2. Open the file in a browser (Chrome recommended)
3. Users will automatically load on page start
4. Press **Reload Users** button to refetch data
5. Turn off internet to test error handling

##  **Output Preview**
- Displays user card with:
  - Name
  - Email
  - Address (street + city)
- Styled responsive layout
- Hover animation
- Error alert if fetch fails
